# cafe wifi website
Website to organize cafe restaurant.

## Features
- Add cafe restaurant
- Filter restaurant by Wifi quality, Socket count, if they have a toilet and if they take calls

## Tech Stack
Python, Flask, Bootstrap5, SQLAlchemy

## Installation
```bash
pip install -r requirements.txt
python main.py
```
website can be reached with [http://127.0.0.1:5000/]()
